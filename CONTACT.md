## 2. Allowed Requests

Allowed inquiries:

- available input data type  
- application domain  
- expected output report type  
- Internal Evaluation License request  
- Internal PoC License request  
- Commercial Integration License request  
- Neotro Certified / Domain Calibration request  
- private repository access request  
- License Telemetry implementation inquiry  
- Registry check inquiry

## 3. Not Accepted

The following requests are outside default collaboration scope:

- request to re-explain or renegotiate basic protocol order  
- request to modify read-only principle  
- request for automatic control function  
- request for free commercial use  
- request for free disclosure of core calibration parameters  
- request for use without attribution  
- request for unauthorized Certified use  
- request for third-party sublicensing  
- request for prompt bypass or circumvention

## 4. Request Form Fields

A valid request should include:

- organization name  
- department  
- responsible contact  
- application domain  
- use purpose  
- intended usage stage  
- input data type  
- output report purpose  
- commercial use status  
- desired license scope  
- administrator notice / usage log support  
- private repository access request
